Hack Me - Challenge by Tech Raj

Rules:

  • No Brute forcing or DDOS attacks allowed. If found, you will be disqualified
  • Challenge duration is 2 weeks from the time the challenge went live.
  • Challenge will end on 18th February
  • If someone solves this challenge, it will go offline, and the webpage will display the name of the person who hacked it.

How to solve:

  • This webapp is made with python django web framework.
  • The goal is to capture a secret key which is stored on the server.
  • The webapp has two functionalities : register and login. First, the user can register for an account, and then he/she can login to their account.
  • If you captured the flag (secret key), submit it here.
